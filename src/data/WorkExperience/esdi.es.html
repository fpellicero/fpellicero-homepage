Trabajé como profesor de desarrollo Frontend a estudiantes del máster de UX/UI. El curso se centraba en el desarrollo de sitios mayoritariamente estáticos, utilizando principalmente HTML y CSS con un poco de Javascript.

Los estudiantes empezaban con poco o ningún conocimiento. Al acabar el curso, podían construir prototipos de alta fidelidad de sus proyectos finales de carrera.

  • Enseñando a estudiantes sin experiencia previa sobre HTML, CSS y Javascript.
  • Introducción a frameworks CSS: Bootstrap, Bulma, Materialize.
  • Introducción a Javascript / jQuery, y como utilizarlos para construir prototiops de alta fidelidad.
